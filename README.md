# Every Day Life

An aesthetic and minimal e-commerce frontend built with Vite + React + TypeScript and Tailwind CSS.

**Live Demo:** [everydaylife.vercel.app](https://everydaylife.vercel.app)

## Features
- Clean, minimalist frontend design
- Fully responsive layout
- Shopping UI components and sample pages

## Local development
Prerequisites: Node.js (18+ recommended) and npm.

Install dependencies:
```bash
npm install
```

Start dev server:
```bash
npm run dev
```

Build for production:
```bash
npm run build
```

Preview production build locally:
```bash
npm run preview
```

The dev server runs by default at http://localhost:8080/.
